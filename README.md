friskWeather 2
日本全国の天気を、速く・美しく・直感的に。

friskWeather 2 は、気象庁・Open‑Meteo・つくみ島 API など複数のデータソースを統合し、
現在の天気 / 体感温度 / 降水確率 / レーダー / 台風情報 を 1 つの画面で確認できるウェブアプリです。

Tailwind CSS と Leaflet を使った軽量 UI で、PC でもスマホでも快適に動作します。

✨ 主な特徴
🗾 全国 47 都道府県に完全対応
都道府県 → 地域 → 市町村 の 3 階層で選択

regionsData により全国の主要地点を網羅

選択変更時は天気カード・レーダーが即更新

📍 IP ベースの自動位置判定
api.ipify.org → ipapi.co を利用

緯度経度から最寄りの観測地点を自動選択

初回アクセスで「現在地の天気」を自動表示

🌤 今日・明日の天気（気象庁 API）
天気アイコン（絵文字）

最高 / 最低気温

降水確率（0–6 / 6–12 / 12–18 / 18–24 時）

詳細天気をテロップとして横スクロール表示

🌡 Open‑Meteo の現在気象データ
現在気温

湿度

風速

風向（角度 → 8 方位に変換）

体感温度（独自計算）

10℃以上：湿度・風速を加味した簡易式

10℃未満：風冷え（Wind Chill）公式

🛰 雨雲レーダー（Leaflet + 気象庁タイル）
気象庁 Nowcast のタイル画像を使用

最新のレーダー時刻を自動取得

再生 / 停止 / 最新に戻る の 3 ボタン

500ms 間隔でアニメーション再生

レイヤーは自動で差し替え

🌀 台風情報（気象庁 XML フィード）
extra.xml から台風情報を取得

タイトル・更新日時を表示

台風がない場合は「現在、台風は発生していません」

🕒 ミリ秒単位のリアルタイム時計
setInterval(10ms) による高精度時計

ヘッダー中央に常時表示

🚀 動作の流れ（内部処理）
IP から現在地を推定

最寄りの観測地点を自動選択

気象庁・Open‑Meteo からデータ取得

今日・明日の天気カードを描画

レーダーマップを初期化

台風情報を取得して表示

地域選択 UI のイベントをセットアップ
